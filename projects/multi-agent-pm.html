多智能体业务咨询助手

Multi-Agent · Business Advisory · Function Calling · Prompt Engineering

该项目面向初创团队的客户咨询、业务介绍和线索转化场景，基于 function calling 与 prompt engineering 构建多智能体业务咨询助手，完成客户演示、场景适配与迭代验证。

Role

深度参与场景拆解、能力编排、提示词方案、工具调用设计和演示链路推进。

Keywords

多智能体 业务咨询 Function Calling Prompt Engineering 客户演示

Background

创业团队的 AI 应用既要讲清产品能力，也要承接问答、推荐、表单收集和后续跟进。相比单一问答助手，多智能体方案更适合表达多角色协作、工具接入和转化路径。

What I Did

  • 基于 function calling 与 prompt engineering 构建多智能体业务咨询助手。
  • 参与场景拆解、能力编排和提示词策略设计，组织产品介绍、问答、推荐、线索收集和工具调用流程。
  • 配合完成客户演示和试点场景适配，推动系统进入可迭代验证状态。

Output

项目形成面向业务咨询和客户沟通场景的多智能体演示链路，验证了多 Agent 能力在初创团队产品展示、需求收集和转化支持中的可行性。
