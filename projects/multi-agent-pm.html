多智能体文旅系统

Multi-Agent · Tourism Scenario · Function Calling · Prompt Engineering

该项目面向文旅展示和服务场景，基于 function calling 与 prompt engineering 构建多智能体文旅系统，在大型行业展会完成展出，并已在相关文旅服务场景推广落地。

Role

深度参与多智能体项目的场景拆解、能力编排、提示词方案和展示链路推进。

Keywords

多智能体 文旅场景 Function Calling Prompt Engineering 展会展示

Background

文旅 AI 应用需要把讲解、问答、推荐、工具调用和展示体验串联起来。相比单一问答助手，多智能体方案更适合表达多角色协作和场景化服务流程。

What I Did

  • 基于 function calling 与 prompt engineering 构建多智能体文旅系统。
  • 参与场景拆解、能力编排和提示词策略设计，推动系统进入可展示状态。
  • 配合完成大型行业展会展示，并推动系统在相关文旅服务场景推广落地。

Output

项目形成面向文旅场景的多智能体展示链路，验证了多 Agent 能力在企业展示和文旅服务场景落地中的可行性。
