# 胡镇涛个人主页

这是一个参考学术个人主页风格制作的纯静态 homepage，可直接托管到 GitHub Pages。

## 文件结构

- `index.html`：主页，包含个人简介、教育经历、能力方向和项目列表
- `styles.css`：页面样式
- `projects/`：项目详情页，包括康养数字人、AI 漫剧、多 Agent 项目管理、视频生成工作流等
- `assets/project-videos/`：主页视频轮播使用的演示片段，目前包含 AI 培训、OpenChatAvatar 数字人、Coze/豆包陪伴型数字人、算力中转和 video-translator 翻译出海演示
- `assets/profile.jpg`：个人照片
- `assets/resume-ai-product.pdf`：AI 产品方向简历
- `assets/resume-research.pdf`：AI 算法方向简历

## 本地预览

直接双击打开 `index.html` 即可。

如果用 GitHub Pages，仓库根目录放这些文件，或者把 `personal-site` 目录内容复制到发布分支根目录。

## 发布前检查

- 确认 PDF 简历中的手机号、住址等信息是否适合公开。
- 不要上传 `person_info` 目录里的证件或银行卡图片。
- 若后续补充项目详情页，建议只放公开安全的项目摘要、架构图和结果，不上传内部文档原件。
